天安门广场一日游

发表于 2017-09-23 | | 阅读次数: 
今天，怀着激动无比的心情来到了北京天安门广场，一出地铁口，
就看到了几座城楼，心里暗自感叹到，哇！毕竟是帝都啊，真壮观！
首先来一张经典的天安门城楼罩，那是曾经毛主席站过的城楼，上图：






接着从地下通道穿过马路来到城楼地下，人实在是太多了！



然后，跟着大部队进入门内，那是人山人海啊，接着想着想上去城楼上
一睹帝都的风采，首先得去城楼购票处购买门票15元一张，还算便宜
（不过真不怎么样，上去不到5分钟就下来了，因为真的没啥好看的！）。
这规定是不让带包的，那我去旅游能不带个包吗？那怎么办？寄包呗，
真是一波三折啊，等城楼5分钟，寄包两小时！哈哈，寄个包还得排队，
关键寄包竟然是人工的，科技这么发达的今天，为什么不搞一个
全自动的呢，寄包浪费了好多时间。
终于要登楼了，登上城楼一看，基本上什么都没有，无奈只好拍了
几张照片就下来了!
这个天下都是朕的！哈哈哈




接着就是去故宫了，不得不说故宫是真的大，走的我腿都酸了。
首先依旧是购票，因为正处于旺季，门票60元一张，价格也还行。
进去看到的便是午门，来人，将此贼犯拖出午门斩首！

检完票，从正中间的大门进入故宫，午门为故宫的入口点。
进去后又是人山人海啊！

依次参观了文华殿，太和门，慈宁宫，乾清宫，御花园等景点，
感觉都是一个样！随便上几张图



最后就是出口了，神武门

还有护城河



一路走下来，太累了，也没拍多少照片，将就看看吧，嘻嘻！
